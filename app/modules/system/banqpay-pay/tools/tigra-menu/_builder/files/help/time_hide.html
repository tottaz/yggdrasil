Hide Delay

Delay in milliseconds between the moment the mouse pointer leaves the menu item and the menu collapses. This delay prevents menu from collapsing when the visitor moves the mouse over gaps between the menu items or menu blocks.

This setting is shared among all menu items on the same level/depth.

By default the delay is inherited from the parent level.

Millisecond is 1/1,000-th of a second. 200 ms is 1/5 sec.
