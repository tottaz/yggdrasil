CONTEXT HELP
Item Caption

Caption is the text displayed in the menu. This text also appears in the builder's tree navigation and in item's path.

The menu can render the HTML code inserted in the item's caption. Most of the HTML tags are welcome (<img>, <table>, <font>). Some tags can interfere with the styles formatting of the menu. Please refer to the product's documentation (section: "inner HTML") if you plan to use complex HTML code in the menu items.
