Level Height

The height of the menu items in pixels. Menu item is generally rectangular bar with the text inside. This parameter sets the vertical size of the bar which should be big enough to accommodate the selected font as well as all paddings and embedded HTML elements (if any).

This setting is shared among all menu items on the same level/depth. In Tigra Menu PRO and Tigra Menu GOLD the height can be set for each item independently.

By default the height is inherited from the parent level
