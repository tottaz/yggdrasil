Block Vertical Offset

Vertical distance in pixels between the origin point (green square on the illustration above) and the menu block (red square). Vertical offset of the root level defines the position of the menu on the page. Use "Menu Ver. Offset" field of "Edit Menu Settings" form to control this parameter. For sub-menus the origin point is the top left corner of the parent item.

Usually vertical offset equals or close to the height of the menu items in the parent level if parent level is horizontal. With vertical parent level the offset should usually be 0 or some small number. Vertical block offset can be either positive (shift down) or negative (shift up) number.

This setting is shared among all menu items on the same level/depth. In Tigra Menu PRO and Tigra Menu GOLD the vertical block offset can be set for each menu block independently.

By default the offset is inherited from the parent level
