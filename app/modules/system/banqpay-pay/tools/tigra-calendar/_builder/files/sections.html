Navigation Bar
START
mode Selects the menu component. Different configuration options will be presented in the builder depending on the product selected.
template Helps create new menu by selecting from the pre designed menu templates.
load Loads the menu configuration previously saved in the common format. Also see "save" section.
EDIT
items Manages the menu items, menu level settings and general menu parameters.
styles Manages style templates - fonts, colors, paddings etc.
preview Shows the results as you build the menu.
FINISH
save Saves the menu in the common (product independent) format.
export Generates the menu configuration for the selected product. This section also provides the instructions on inserting the menu on the pages.
